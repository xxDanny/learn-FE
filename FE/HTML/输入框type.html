    # HTMl 输入类型

    取值 作用 语法 img
    text 定义用于文本输入的单行输入字段
    image 定义图像形式的提交按钮 必须把 src 属性和 alt 属性与 <input type="image"> 结合使用。 <input type="image" id="image" alt="Login" src="/media/examples/login-button.png">
    reset 定义重置按钮。重置按钮会清除表单中的所有数据。
    password 定义密码字段
    submit 定义用于向表单处理程序 (form-handler) 提交表单的按钮。
    radio 定义单选按钮 <input type="radio" id="huey" name="drone" value="huey" checked><label for="huey">Huey</label>
    checkbox 定义复选按钮
    button 定义按钮
    hidden 定义隐藏字段。隐藏字段对于用户是不可见的；但是提交表单时依旧存在
    file 用于文件上传。multiple 属性设置一次可以上传多个文件，required 属性规定必须上传一个文件。使用该 accept 属性来定义控件可以选择的文件类型。 <input type="file" id="avatar" name="avatar" accept="image/png, image/jpeg">

    # HTML5 输入类型

    type 作用
    email
    search 搜索字段旨在用于在页面和应用程序上创建搜索框
    tel 当通过带有动态键盘的触摸设备进行访问时，大多数设备会在 type="tel"遇到时显示数字小键盘
    url 它向字段添加了特殊的验证约束。如果没有输入协议（例如 http:），或者 URL 格式不正确，浏览器将报告错误
    number 用于输入数字的控件
    range 另一种选择数字的方法是使用滑块
    datetime-local 创建一个小部件来显示和选择一个没有特定时区信息的带时间的日期。
    month 创建一个小部件来显示和选择带有年份的月份
    time 创建一个小部件来显示和选择时间值。虽然时间可能以 12 小时格式显示，但返回的值是 24 小时格式。
    week 创建一个小部件来显示和选择周数及其年份
    date 所有日期和时间控件都可以使用 min 和 max 属性进行约束，还可以通过 step 属性进行进一步的约束（其值根据输入类型而变化）。
    color 返回的值始终是小写的 6 值十六进制颜色。
