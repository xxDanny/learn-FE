    农业银行

    # 农业银行

    [蓝蓝银行系列]2 农行初探+笔试形式+面试技巧=上岸](https://www.nowcoder.com/discuss/730735) 【经验贴】农业银行软开面试资料之查漏补缺！看看不足在哪里 (opens new window) 农业银行笔试题型｜记录贴 (opens new window) 2021 中国农业银行 笔试 (opens new window) 中国农业银行西安研发中心 2022 春招 (opens new window) ​第七击 | 农行这些事儿 (opens new window) 求农行陕西省分行-西安分行信息技术岗后续通知 (opens new window) 面试复盘|2022 届 农行研发中心笔试第一批+面经 (opens new window)

    笔试是上机做题，大概分为数学逻辑+英语阅读理解+计算机基础（JAVA+C 语言+计算机网络+操作系统+数据库）等几部分，类似于计算机考研专业课的东西，再加个心理测试（不计入总分）

    1、n 个体检项目，体检项目之间存在先后关系，每天可以完成任意数量体检项目，输出完成全部项目所用的天数 2、机器人位置移动，避障，判断是否能完成任务 3、字符串数组形式的数据包，判断第 i 行和第 i 列队迎的字符串是否相同

    第一道是三进制，第二道是找质点，第三道是对扑克牌排序

    java c c++ python

    一共 3 小时，四个模块分别计时。 分别是常识大乱炖、行测、英语（选词填空➕四篇阅读理解）、计算机基础（数据库 Java c 测试 数据结构）➕代码选择大乱炖

    （1）客观题 70 道共 60 分钟：25 道是数据库相关知识，包括数据库相关概念、基础知识、sql 语句等；10 道数据结构；10 道计算机网络；15 道给代码写结果和程序填空（java 和 c）；10 道行测题（数量关系） （2）编程题 3 道共 90 分钟： 现有一只股票 n 天的收盘价数据，编程计算 k 日均线数据，k 日均线数据包含 n-k+1 个数据点。题目给一个长度为 20 的数组，k 设定为 5。 现有一个 5 行 5 列的二维列表，每个值代表 5 个人之间的支持度，要求找出 5 个人中从其他人那里得到最大支持的人的序号，如果最大支持数相同，返回编号最小的序号。 现有三名同学，均选修了 5 门课程，按照总分降序输出三个同学的姓名。输入值包括姓名和得分两个数组。 （3）语言的话可以用 c、c++、java、Python ，需要自己处理输入输出。tips：最好记住一些字符串转换的 api

    1，股票 k 日均价（大概是滑动窗口的平均值） 2，支持度（矩阵按列求和，然后排序） 3，按总分排序（字典排序）
